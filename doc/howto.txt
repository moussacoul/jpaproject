How to Use
-------------

1- Connectez vous avec votre identifiant via la main page ou créer votre compte puis connectez vous.

2- Customisez votre CV en y ajoutant vos formations, expériences ...

3- Share.


Fonctionnalités
-------------

Login - Register : Connexion d'un compte avec un identifiant et un mot passe. Création d'un compte.

Alimenter votre profil : En naviguant sur les pages via les boutons, nous pouvons ajouter ou supprimer : centre d'intérêts, compétence, expérience, formation, informations personnelles, langue.

Gestion de la session : Un utilisateur est connecté en permanence après la connexion.

JPA : Persistance des données gérer avec l'interface JPA pour gérer la liaison avec la base de données.

AJAX : Gérer automatiquement avec BootFaces.

Design : Utilisation de Bootstrap via BootFaces.

JavaServer Faces : Utilisation de l'état des composants.
